About the Stress Detection System

This project is designed to help IT professionals understand and manage their stress levels using intelligent machine learning techniques. By analyzing behavioral data such as workload, work hours, and task pressure, the system predicts whether a user’s stress level is Low, Moderate, or High.

At its core, the system uses a stacked ensemble machine learning model that combines the strengths of Random Forest, AdaBoost, Extra Trees, and SVM to deliver accurate and reliable predictions.

When high stress is detected, the system provides alerts, guided meditation videos, and useful links to counseling and mental health support platforms.

This tool promotes a healthier work environment — because your mental well-being matters.
