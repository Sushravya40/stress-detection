About the Stress Detection System

This project is designed to help IT professionals understand and manage their stress levels using intelligent machine learning techniques.

By analyzing behavioral data such as workload, work hours, and task pressure, the system predicts whether a user's stress level is Low, Moderate, or High.

The system uses a stacked ensemble model combining Random Forest, AdaBoost, Extra Trees, and SVM to deliver accurate predictions.

When high stress is detected, helpful resources and guidance are provided to support mental well-being.
